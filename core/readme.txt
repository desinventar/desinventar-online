
DesInventar8 - Core Module

Requeriments
------------
- Java JDK or JRE installed (java and javac should be added to the PATH)
- Ostermiller Utils 
- Apache XMLRPC
- MySQL J-Connector
- MySQL 4.1.20 or newer
- SQLiteJDBC


MySQL Configuration
-------------------
- Grant dicore user all permissions needed to create new databases
  for regions.

  GRANT ALL ON di8db.* TO di8db@localhost identified by 'di8db';
