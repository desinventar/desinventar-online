How to create a Windows Installer for this Application

- Install.nsi is the install script which will build the
  installer in the 'setup' directory, all other installer 
  files must be in that directory.

Prepare the enviroment
----------------------
- Install NSIS compiler system (http://nsis.sourceforge.net)
- Install HM NIS Edit (http://hmne.sourceforge.net)
- Install the TextReplace plugin for NSIS, download from
  http://nsis.sourceforge.net/TextReplace_plugin
- Install the ZIPDll plugin for NSIS, download from
  http://nsis.sourceforge.net/ZipDLL_plug-in
