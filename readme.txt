Requeriments
------------
- PHP (php-pdo php-xml php-gd)
- MapServer
- SQLite

PHP Components
- Smarty-2.6.26 (http://smarty.php.net/)
- Jpgraph-3.0.3 (http://www.aditus.nu/jpgraph/)
- arial.ttf must be installed, more info at:
  http://corefonts.sourceforge.net/

Javascript Components
- ExtJS 2.3.0 (http://www.extjs.com/) (don't use ExtJS 3.X)
- OpenLayers.org (http://www.openlayers.org/) (tested with version 2.11)
