Requerimientos de Instalación (Linux)
-------------------------------------
- Apache Web server
- PHP 5.3 (o 5.4) Instalar modulos GD, XML, PDO, dbase
- mapserver (5.6.7)

Librerias PHP
-------------
Smarty
jpgraph-3.5.0b1

Librerias JS
------------
jQuery (1.8)
jQuery UI (1.8)
jQuery module : bgiframe
jQuery module : dimensions
jQuery module : metadata
jQuery module : tooltip
jQuery module : bbq
jQuery module : http://jqueryplugins.blogspot.com/
Valum's AJAX File-Uploader
extJS (3.4.0)
OpenLayers (2.11)

Configuración Básica
--------------------
- Carpeta para cache de Smarty
  /var/cache/Smarty/desinventar
- Archivos temporales
  /var/tmp/desinventar
- Archivos bases de datos
  /var/lib/desinventar
- Archivos de acceso via web
  /var/www/desinventar/graphs
- Fuente de DesInventar
  /usr/share/desinventar
- Mapas bśicos (worldmap)
  /usr/share/desinventar/worldmap
